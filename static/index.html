🚀 Cara Penggunaan & Metadata

🏷️ Tagging Placeholder

Gunakan format {{ variable }} di Word untuk data manual/profil (e.g. {{ nomor_sk }}). Untuk isian otomatis oleh AI, gunakan awalan ai_ seperti {{ ai_pembukaan }}.

🧠 Konteks & Regulasi

Paste standar Akreditasi atau Pokok Pikiran ke dalam kotak Konteks. AI akan merancang isi SOP/SK agar 100% selaras dengan regulasi tersebut.

🔄 Workflow

1. Upload template Word ber-tag.
2. Input Topik & Konteks.
3. Klik Magic Fill & Review.
4. Generate Docx siap pakai.

📊 Advanced: Tabel & List Dinamis

Tabel Prosedur (SOP):

• Awal: {% tr for itm in items %}
• Isi: {{ itm.langkah }}
• Tutup: {% tr endfor %}

Diktum/List (SK):

• Awal: {% p for d in ai_menetapkan %}
• Isi: {{ d }}
• Tutup: {% p endfor %}
*Ini akan membuat KESATU, KEDUA dst. secara dinamis.
